# ☕ Aesthetic Coffee Data Analysis App

A simple and aesthetic Python project that analyzes coffee sales data using data science libraries and visualizes insights with beautiful charts.

---

## 🚀 Features

* 📊 Coffee sales dataset using Pandas
* 📈 Data analysis (average price, total sales)
* 📉 Bar chart using Matplotlib
* 🥧 Interactive pie chart using Plotly
* 🔥 Automatically finds most popular coffee

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Plotly

---

## 📂 Project Structure

```
coffee-data-app/
```

## ▶️ How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the app:

```
python coffee_app.py
```
